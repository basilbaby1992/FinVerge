# FinVerge

FinVerge is an enterprise-grade microservices-based reservation and inventory management platform built with Spring Boot.

## Technology Stack

- Java 17
- Spring Boot
- Spring Cloud Gateway
- Spring Security
- JWT
- OpenFeign
- Kafka
- Eureka
- MySQL
- Docker (later)
- Kubernetes (later)

## Services

- API Gateway
- Auth Service
- Inventory Service
- Booking Service
- Notification Service
- Common Library
